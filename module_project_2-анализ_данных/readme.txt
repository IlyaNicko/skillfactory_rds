Задачи проекта:

Применение библиотек Pandas, matplotlib, numpy, seaborn
Очистка и подготовка данных для анализа
Освоение статистической обработки данных
Корреляционный анализ данных
Применение навыков работы с DataFrame
Закрепление навыков работы с различными типами величин
Продолжение изучения возможностей Python

Данные представлены датафреймом из 30 столбцов и 395 строк. Есть пропуски, данные, заполненные нулевым значением, ошибочные введения.
